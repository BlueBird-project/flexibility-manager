FM.optimize — Function

Run the MPC for 1 time step considering the linear model 

optimize(A, B, F, x, d=0 ;
         x_low=nothing, x_high=nothing,
		 u_low=nothing, u_high=nothing,
		 kwargs...)
			  

``x_{k+1} = A x_k + B u_k + G d_k``

Input  : 
A  State Matrix
B  Input Matrix
G  Disturbance/Forecast Matrix 
x  Current State used as MPC initial condition 
d  Disturbances (forecasts). 
x_low   Low bound constraints on state (To be defined for each state, used Inf if no constraint)
x_high  High bound constraints on state (To be defined for each state, used Inf if no constraint)
u_low   Low bound constraints on action (To be defined for each state, used Inf if no constraint)
u_high  High bound constraints on action (To be defined for each state, used Inf if no constraint) 


Output :
Control Action. For example temperature setpoints for HVAC control
